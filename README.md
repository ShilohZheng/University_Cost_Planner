# University Cost Planner

A modern and interactive finance/accounting website created as a Grade 12 Capstone Project.

## About the Project

This website helps high school students estimate and compare the costs of attending university in Canada. The project focuses on financial literacy, budgeting, and helping students make informed financial decisions before post-secondary education.

## Features

- University cost comparison
- Budget calculator
- Savings goal tracker
- Financial literacy tips
- Scholarship information
- Interactive charts and graphs
- Responsive mobile-friendly design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Chart.js
- GitHub Copilot

## Purpose

The purpose of this project is to combine finance/accounting concepts with web development skills to create a practical tool for students planning their future education.
